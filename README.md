# DIELCA Commercial War Room 2026

Mini app web para dinamizar el workshop JetSelling® Advanced Lab 2026.

## Publicación rápida en GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube `index.html`, `styles.css` y `app.js` a la raíz del repositorio.
3. Ve a Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Guarda y espera a que GitHub genere la URL.

## Uso en sala

- Comparte el enlace o un QR.
- Cada equipo selecciona su nombre.
- Las respuestas se guardan localmente en el navegador.
- Al final, cada equipo pulsa “Copiar resumen” o “Descargar respuestas”.
