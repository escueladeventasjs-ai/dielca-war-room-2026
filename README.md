# DIELCA War Room Game V3

Versión gamificada móvil.

## Características

- Casi sin escritura.
- Selección por chips.
- Misiones.
- Puntuación.
- Resumen copiable.
- Pensada para móvil durante el workshop.

## Instalación

Sustituye en GitHub los archivos:
- index.html
- styles.css
- app.js
- README.md

Luego espera a que GitHub Pages actualice.
